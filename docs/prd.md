# 选择困难症救星网站需求文档

## 1. 网站名称
选择困难症救星

## 2. 网站描述
一个帮助用户快速做出生活决策的趣味网站，解决吃什么、周末做什么、追什么剧、看什么书的选择难题。采用抽盲盒的互动形式，增加趣味性和随机性。

## 3. 网站功能
### 3.1 美食推荐
用户摇一摇手机或点击摇动按钮，触发盲盒晃动动画，随机显示提示信息（如：辣的、清淡的、甜的、咸的等口味提示）。用户点击盲盒按钮，触发开盒动画后随机推荐一种美食类型或具体菜品

### 3.2 周末活动推荐
用户摇一摇手机或点击摇动按钮，触发盲盒晃动动画，随机显示提示信息（如：室内、户外、休闲、运动等类型提示）。用户点击盲盒按钮，触发开盒动画后随机生成周末活动建议，如户外运动、看电影、逛展览、宅家休息等

### 3.3 影视剧推荐
用户摇一摇手机或点击摇动按钮，触发盲盒晃动动画，随机显示提示信息（如：悬疑、喜剧、爱情、科幻等类型提示）。用户点击盲盒按钮，触发开盒动画后随机推荐热门或经典影视剧，包含剧名和简短介绍

### 3.4 书籍推荐
用户摇一摇手机或点击摇动按钮，触发盲盒晃动动画，随机显示提示信息（如：小说、传记、科普、心理等类型提示）。用户点击盲盒按钮，触发开盒动画后随机推荐书籍，涵盖不同类型，包含书名和推荐理由

### 3.5 一键分享功能
用户获得推荐结果后，点击分享按钮，弹出精致排版的分享图片弹窗。图片采用卡片式设计，包含推荐内容、网站标识、装饰性图标元素，配色与网站主题一致，排版层次分明。用户可长按保存图片或直接分享到社交平台

## 4. 交互动画效果
- 摇一摇触发时，盲盒产生左右晃动和轻微旋转动画，同时在盲盒上方显示提示文字（带渐入效果）
- 盲盒采用3D立体盒子造型，具有明显的盒盖、盒身和阴影效果，增强神秘感
- 点击盲盒时带有缩放和弹跳动画
- 开盒过程展示盒盖向上打开的3D翻转动画，伴随光效和结果弹出效果
- 结果展示时带有渐入和轻微弹跳动画
- 分享按钮点击时带有脉冲动画反馈，弹窗以缩放渐入方式出现

## 5. 网站设计风格
- 配色方案：明快的橙黄色为主色调，搭配清新的浅蓝色和白色背景，营造轻松愉悦的氛围
- 视觉细节：盲盒采用3D立体盒子设计，具有明显的盒盖、盒身结构和投影效果，开盒动画流畅自然，图标采用扁平化风格，分享弹窗图片采用圆角卡片设计，带有细腻阴影和装饰性边框
- 整体布局：响应式卡片布局，手机端采用单列垂直排列，四个功能模块以卡片形式展示，每个卡片居中展示立体盲盒和结果区域，确保手机端操作便捷且视觉舒适
